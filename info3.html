GIT EN EL SERVIDOR

A pesar de que se puede gestionar un propio servidor de Git, existen herramientas mas modernas, con todas las funciones y otras adicionales que pueden ser útiles para el desarrollo de aplicaciones.

Entre las opciones de servidores Web de Git que tenemos están las siguientes:

GitLab

Es una aplicación web, de la cual puede descargarse una máquina virtual, o un instalador para la creación de un propio servidor moderno de control de versión. Aún así existe la versión web, la cual contiene varias características gratuitas como son aquellas que posibilitan la integración continua mientras se realiza el desarrollo de las aplicaciones.

GitHub

Es un servicio para alojamiento de repositorios, que funciona de manera similar que GitLab, pero que no permite la creación de repositorios privados. Su sistema es comercial, a diferencia de Git que es proyecto de código abierto, para acceder a la funcionalidad anteriormente descrita es necesario el pago de mensualidades, aunque es gratuito para alojar proyectos Open Source.
